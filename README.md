# AlphaBeta

Pure-SwiftUI iPhone/iPad app for exploring non-Latin alphabets. Ships with Greek; built to grow (Cyrillic, Arabic, Hebrew, and more).

- **SPEC.md** — full build specification. Start here (Claude Code: follow the milestones in §10).
- **Manifest.json** — language registry (9 languages); ships in the app bundle.
- **Greek / Russian / Ukrainian / Belarusian / Serbian / Bulgarian / Macedonian / Armenian / Georgian .json** — alphabet content (schema v2); ship in the app bundle.
- **Greek-V1.json** — retired flat-key schema v1, kept for reference; not bundled.

Legacy UIKit prototype: sibling repo `../AlphaBetaV1`.
